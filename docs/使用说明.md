# iRecord 使用说明

## 简介

iRecord（你说我记）是一款本地离线语音转写桌面工具，支持录音转写和音视频文件转写，所有语音识别在本地完成，不上传任何数据。同时集成大模型分析功能，可对转写结果进行摘要、纪要、问答等深度处理，并整理成结构化知识文档。

---

## 安装与配置

### 1. 安装软件

运行 `iRecord-Setup-x.x.x.exe` 安装包，按提示完成安装。

### 2. 准备资源文件

软件需要以下资源文件，放置在 `resources/` 目录下：

| 目录 | 说明 | 大小（约） |
|------|------|-----------|
| `models/sherpa-onnx-sense-voice-zh-en-ja-ko-yue-2024-07-17` | SenseVoice 模型（默认内置，轻量快速，中英日韩粤） | ~240MB |
| `models/sherpa-onnx-qwen3-asr-0.6B-int8-2026-03-25` | Qwen3-ASR 模型（可选，高精度） | ~950MB |
| `models/silero-vad` | VAD 语音活动检测模型 | ~1MB |
| `models/speaker-diarization` | 说话人分离模型 | ~45MB |
| `ffmpeg/ffmpeg.exe` | 音视频格式转换工具（处理视频/非标准音频时需要） | ~140MB |

SenseVoice 等默认模型随软件提供。Qwen3-ASR 体积较大，可在 **设置 → 模型管理** 中按需下载。

### 3. 检查模型状态

进入 **设置 → 模型管理**，可查看各模型的安装状态。未安装的可选模型（如 Qwen3-ASR）可在此直接下载或删除。

---
## LLM 智能分析配置

大模型分析功能（摘要、纪要、问答等）和知识整理需要配置 LLM API。

进入 **设置 → LLM 配置**：

- **服务商**：阿里百炼（DashScope）或 DeepSeek
- **模型**：
  - DashScope：`qwen3.7-max` / `qwen3-max` / `qwen3.6-plus` / `qwen3.5-flash`
  - DeepSeek：`deepseek-v4-flash` / `deepseek-v4-pro`
  - 也可输入自定义模型代码（如 `qwen-plus-latest`）
- **API Key**：填入对应服务商的 API Key（每个服务商需单独配置）

**获取 API Key：**

- 阿里百炼：访问 [阿里云百炼平台](https://bailian.console.aliyun.com/)，开通服务后在 API Key 管理页创建密钥
- DeepSeek：访问 [DeepSeek 开放平台](https://platform.deepseek.com/)，在 API Keys 页面创建密钥

> 不配置 API Key 也可正常使用语音转写，但 AI 分析和知识整理功能不可用。

## 功能说明

主界面分为三个标签页：**录音**、**文件上传**、**知识整理**。

### 录音转写

1. 在「录音」标签页点击开始录音
2. 录音过程中可随时停止；停止后弹出确认，可选择「停止录音」「继续录音」或「不保存」
3. 停止后可勾选「创建语音转写」，再点击「保存录音」
   - 勾选时：保存音频并自动创建本地离线转写任务，可在录音列表查看进度
   - 不勾选时：仅保存音频记录
4. 转写完成后，点击记录进入详情页查看结果

> 说明：当前版本为「录音后处理」模式，录音时不做实时转写，停止后再统一离线识别，准确率更高。

### 文件转写

1. 在「文件上传」标签页，点击添加文件或直接拖放音视频文件
2. 软件自动排队处理，完成后点击任务查看转写结果
3. 支持格式：音频 WAV、MP3、FLAC、M4A、AAC、OGG；视频 MP4、MKV、AVI、MOV、FLV（自动提取音轨）

### 转写分段策略

在 **设置 → 文件识别** 中可选择分段策略：

- **自动选择（推荐）**：根据音频自动决定
- **说话人分离**：区分多人对话并标注说话人
- **VAD 分段**：按语音活动检测切分
- **整体识别**：不分段，整段识别

### AI 智能分析

转写完成后，在结果详情页可使用大模型分析功能：

- **摘要**：提取核心要点
- **发言人**：按说话人归纳各自观点
- **纪要**：生成结构化会议纪要
- **问答**：从内容中提取问答对
- **提问**：针对内容自由提问

> 需先配置 LLM API Key。

### 知识整理

1. 在「知识整理」标签页，点击「新建文档」
2. 选择模板和来源转写记录
3. 大模型自动生成结构化文档
4. 在内置富文本编辑器中可二次编辑，支持字数统计
5. 导出为 Markdown / TXT / PDF

### 说话人分离

选择说话人分离策略时，软件自动检测多人对话并标注说话人，无需手动设置人数。结果页按说话人区分显示。

### 导出

- 复制全文到剪贴板
- 录音：导出为 WAV
- 转写任务：导出为 TXT（可选含时间戳/说话人标签）
- 知识文档：导出为 Markdown / TXT / PDF

- 建议内存 4GB 以上

## 常见问题

**Q：识别速度慢怎么办？**
A：软件使用 CPU 推理。默认的 SenseVoice 模型速度较快；追求精度可切换 Qwen3-ASR，但耗时更长。暂不支持 GPU 加速。

**Q：没有配置 LLM API Key 能用吗？**
A：可以正常使用语音转写功能，但 AI 分析（摘要、纪要等）和知识整理功能不可用。

**Q：支持哪些语言？**
A：SenseVoice 支持中、英、日、韩、粤五种语言；Qwen3-ASR 支持更多语言，精度更高。

**Q：录音时为什么看不到实时字幕？**
A：当前版本采用录音后处理模式，录音结束后再统一离线转写，不提供实时字幕，以换取更高的识别准确率。
