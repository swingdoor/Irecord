# iRecord 使用说明

## 简介

iRecord（你说我记）是一款本地离线语音转写桌面工具，支持文件转写和实时录音识别，所有语音识别在本地完成，不上传任何数据。同时集成 AI 智能分析功能，可对转写结果进行摘要、纪要、问答等深度处理。

---

## 安装与配置

### 1. 安装软件

运行 `iRecord-Setup-x.x.x.exe` 安装包，按提示完成安装。

### 2. 下载资源文件

安装完成后，需要额外下载以下资源文件：

**下载地址：** [待补充]

下载内容包括：

| 目录 | 说明 | 大小（约） |
|------|------|-----------|
| `models/sherpa-onnx-qwen3-asr-0.6B-int8-2026-03-25` | Qwen3-ASR 模型（高精度，52 语言） | ~600MB |
| `models/sherpa-onnx-sense-voice-zh-en-ja-ko-yue-2024-07-17` | SenseVoice 模型（轻量快速，中英日粤韩） | ~200MB |
| `models/sherpa-onnx-streaming-zipformer-zh-int8-2025-06-30` | Zipformer 流式模型（实时录音用） | - |
| `models/silero-vad` | VAD 语音活动检测模型 | - |
| `models/speaker-diarization` | 说话人分离模型 | - |
| `ffmpeg/ffmpeg.exe` + `ffprobe.exe` | 音频格式转换工具 | ~140MB |

### 3. 配置资源目录

首次启动软件后，进入 **设置 → 基础设置**，配置以下路径：

- **模型目录**：指向下载的 `models` 文件夹所在目录
- **FFmpeg 目录**：指向下载的 `ffmpeg` 文件夹路径

配置完成后软件即可正常使用。

## LLM 智能分析配置

AI 分析功能（摘要、纪要、问答等）需要配置大模型 API：

进入 **设置 → LLM 配置**：

- **模型提供商**：阿里百炼（DashScope）
- **模型选择**：`qwen3-max`（推荐）/ `qwen3.6-plus` / `qwen3.5-flash`
- **API Key**：填入阿里云百炼平台的 API Key

**获取 API Key：**

1. 访问 [阿里云百炼平台](https://bailian.console.aliyun.com/)
2. 注册/登录阿里云账号
3. 开通百炼服务，在 API Key 管理页面创建密钥
4. 将密钥粘贴到软件设置中

> 后续版本将支持 DeepSeek 等更多模型提供商。

## 功能说明

### 文件转写

1. 在主页「文件上传」标签页，点击添加文件或直接拖放音视频文件
2. 选择 ASR 模型（Qwen3-ASR 或 SenseVoice）
3. 软件自动排队处理，完成后点击任务查看转写结果
4. 支持格式：WAV、MP3、FLAC、M4A、AAC、OGG、WMA、MP4、MKV、AVI、MOV 等

### 实时录音

1. 在主页点击「录音」或使用全局快捷键 `Ctrl+Shift+R`
2. 开始录音后实时显示转写文字
3. 支持两种引擎：
   - **Qwen3-ASR + VAD**：高精度，延迟 1-2 秒
   - **Zipformer**：低延迟，适合实时场景
4. 录音结束后自动保存结果

### AI 智能分析

转写完成后，在结果详情页右侧可使用 AI 分析功能：

- **摘要**：提取核心要点
- **发言人**：按说话人归纳观点
- **纪要**：生成结构化会议纪要
- **问答**：自动提取问答对
- **提问**：针对内容自由提问

> 需要先配置 LLM API Key 才能使用此功能。

### 知识整理

1. 在主页「知识整理」标签页，点击「新建文档」
2. 选择模板和来源录音/转写
3. AI 自动生成结构化文档
4. 支持编辑、导出为 Markdown / TXT / PDF

### 说话人分离

软件自动检测多人对话并标注说话人，无需手动设置人数。结果页按说话人着色显示，并统计每人发言段数和时长。

### 导出

- 复制全文到剪贴板
- 导出为 TXT（含时间戳和说话人标签）
- 导出为 Markdown / PDF（知识文档）

## 常见问题

**Q：识别速度慢怎么办？**
A：软件使用 CPU 推理（默认 4 线程）。可选择 SenseVoice 模型获得更快速度，或等待后续 GPU 加速版本。

**Q：没有配置 LLM API Key 能用吗？**
A：可以正常使用语音转写功能，但 AI 分析（摘要、纪要等）和知识整理功能不可用。

**Q：支持哪些语言？**
A：Qwen3-ASR 支持 52 种语言，SenseVoice 支持中、英、日、粤、韩五种语言。
